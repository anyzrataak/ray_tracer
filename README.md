# Ray Tracer Project

This project implements a basic ray tracing algorithm to render 3D scenes. It focuses on calculating light physics, including reflections and shadows, for various geometric primitives.

## Technical Specifications
- language: C++
- additional library: SDL3

## Setup and Installation
- configure your compiler to include the include/ directory.
- link the SDL3.lib file from the lib/ directory.
- build and run the project.

## Current Features
- SDL3 window initialization and event handling.
- Vector mathematics and ray-primitive intersection logic.
- Basic primitive rendering.
